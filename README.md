# WEATHERLY

Minimal static weather app built with HTML, CSS, and vanilla JavaScript.

## Features

- Search current weather by free-text city and country
- Enter an OpenWeatherMap API key directly in the page
- View temperature, condition, humidity, and wind speed
- Responsive minimalist interface for mobile and desktop

## Run

Open `index.html` in a browser.

## API

This app uses the OpenWeatherMap Current Weather endpoint:

`https://api.openweathermap.org/data/2.5/weather`

You will need your own API key.
